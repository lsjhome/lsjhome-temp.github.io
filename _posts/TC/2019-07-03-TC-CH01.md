# Chapter 01 프로그램 만들기

## 1.1 프로그램과 C언어

- 프로그래밍의 기본 절차
  저장공간 확보 -> 입력 -> 처리 -> 출력
- C의 특징
  C는 이식성, 함수 사용, 시스템 프로그래밍 가능
- 표준화 과정

## 1.2 컴파일러 사용법

- 프로그램 작성 순서
  소스파일 작성 -> 컴파일 -> 실행
- 소스파일
  아스키코드(ASCII code)에 따라 저장된 파일
- 실행파일
  컴퓨터가 이해할 수 있는 기계어 형태의 파일
- 프로젝트 생성 -> 소스파일 추가 -> 빌드 -> 실행
